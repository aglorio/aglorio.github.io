UX/UI

Huerta de Hierro/Sitio Web

Para este encargo, se pidió diseñar el Sitio Web principal para Huerta de Hierro, un centro cultural medieval en Lima. Se acordó, en la primera reunión, utilizar una plantilla web para la misma, pero crear una interface diferente y temática para dar algo nuevo a la marca.

Huerta de Hierro, además, ofrece una gran gama de servicios y actividades, y cada una de las cuales necesitaba diferenciarse y colocar información clara en distintas secciones de la web. Es asi que se empezó el proyecto mediante la definición de estos servicios, sus alcances reales y los objetivos que tenían al presentarlos en su nueva web.

Alcance y Objetivos

Debido a la complejidad de los servicios ofrecidos por HdH, el sitio se dividió en cuatro subdominios principales, cada uno de los cuales con sus propias ofertas y funciones; además de un objetivo principal para cada una.

Desarrollo

Como el proyecto dependía del uso de una plantilla, el enfoque de diseño fue colocado en la generación de una imágen congruente con la marca, y en la facilidad de uso y navegación para nuevos clientes potenciales, junto con el ofrecimiento de información de calidad sobre lo que ofrece HdH.

Los lineamientos de diseño fueron seguidos fielmente, pero ciertos nuevos aspectos fueron agregados para complementar el estilo medieval fantástico, tomando inspiración de pergaminos y arte perteneciente al medioevo y de páginas web con temáticas medievales de los 90s, fusionando el aspecto histórico con el aspecto nostálgico digital.
Páginas de referencia: Tales of the Middle Ages y The Great Lakes Medieval Faire.
Como no hacía falta la creación de wireframes como tales, las pruebas de descarte con el cliente se realizaon con variaciones de paletas de color, estilos de fotografía y otros elementos de diseño.
Las dos opciones presentadas, cada una con ciertos elementos representativos.

Resultados

El estilo escogido finalmente fue una combinación de elementos de ambas opciones, como un color y textura de fondo claros y sutiles, pero reminicente de un pergamino, fotos oscuras y cálidas, botones de acción llamativos y separadores ornamentales entre secciones clave.
